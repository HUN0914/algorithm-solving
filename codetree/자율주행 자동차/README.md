## 📝 문제 설명
n×m 크기의 도로에 1×1 크기의 자율주행 자동차가 주어집니다.

자율주행 자동차는 다음 규칙에 따라 움직입니다.

1. 현재 방향을 기준으로 왼쪽 방향으로 한 번도 간 적이 없다면, 좌회전한 뒤 해당 방향으로 1칸 전진합니다.
2. 만약 왼쪽 방향이 인도이거나 이미 방문한 도로라면, 좌회전하고 다시 1번 과정을 시도합니다.
3. 2번 과정을 4방향 모두 확인했는데도 전진하지 못한 경우에는, 바라보는 방향을 유지한 채 한 칸 후진한 뒤 다시 1번 과정을 시도합니다.
4. 3번 과정을 시도하려 했지만 뒷공간이 인도여서 후진조차 하지 못한다면 작동을 멈춥니다.

자율주행 자동차가 작동을 멈췄을 때, 거쳐간 도로의 총 면적을 구하는 프로그램을 작성하세요.  
처음 위치의 도로도 총 면적에 포함합니다.

---

## 📥 입력
첫 번째 줄에 도로의 세로 크기와 가로 크기를 의미하는 정수 n과 m이 주어집니다.

두 번째 줄에 자율주행 자동차의 초기 위치 (x, y)와 바라보는 방향 d가 주어집니다.  
d는 0부터 3까지의 정수로 주어지고, 순서대로 북쪽, 동쪽, 남쪽, 서쪽을 의미합니다.  
자동차의 처음 위치 x는 위쪽에서부터 아래쪽까지 0부터 n-1까지 차례대로 번호를 매겨 구분하며,  
y는 왼쪽에서 오른쪽까지 차례대로 번호를 매겨 구분합니다.

세 번째 줄부터 n+2번째 줄까지는 도로의 상태가 주어집니다.  
도로는 0 또는 1의 값을 가지며 0은 도로, 1은 인도를 의미합니다.

- 자율주행 자동차가 있는 칸은 도로일 것이라 가정해도 좋습니다.
- 격자의 첫번째 행과 마지막 행, 첫번째 열과 마지막 열은 항상 인도일 것이라고 가정해도 좋습니다.

---

## ⚠️ 제한 조건
- 3 ≤ n, m ≤ 50
- x와 y는 정수입니다.

---

## 📤 출력
초기 상태의 방문 면적을 1로 가정하고, 자율주행 자동차가 방문했던 도로의 총 면적을 구합니다.

## 내가 틀린이유 
북쪽은 dy값이 1이 아니라 -1이다.
