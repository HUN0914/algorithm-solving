📝 문제 설명

2^n × 2^n 격자의 빙하를 회전시키려고 합니다.

빙하에 속한 각각의 얼음은 인접 회전성이라는 성질을 가집니다. 좌측 상단에 있는 빙하를 특정 범위를 선택해서 회전시키게 되면 다른 상하좌우의 인접한 격자들도 똑같은 크기만큼 회전하게 됩니다.

빙하를 회전하는 범위를 레벨이라고 표현합니다. 레벨이 L일 때 2^L × 2^L 만큼 격자를 선택하여 2^(L−1) × 2^(L−1) 만큼 잘라 4등분하여 회전시키게 됩니다. 예를 들어 격자에 레벨 1과 2만큼 순서대로 회전을 진행하면, 레벨 1 회전 후의 상태를 기준으로 다시 레벨 2 회전이 진행됩니다. 레벨 0의 경우 회전이 존재하지만, 실제로는 위치 변화가 없습니다.

각각의 회전이 모두 끝나고 난 뒤에는 빙하에 속한 얼음이 녹습니다. 한 칸을 기준으로 상하좌우 인접한 칸에 얼음이 3개 이상 있는 경우에는 녹지 않습니다. 그렇지 않은 경우에는 1이 줄어듭니다. 인접한 칸이 격자를 벗어나는 경우나 해당 칸의 값이 0인 경우에는 얼음이 존재하지 않는다고 봅니다. 얼음이 녹는 것은 동시에 진행됩니다.

모든 회전을 끝내고 난 뒤에 남아있는 빙하의 총 양과 가장 큰 얼음 군집의 크기를 출력해야 합니다. 빙하의 총 양은 격자에 남은 숫자의 총합이며, 얼음 군집은 상하좌우로 연결된 칸들의 집합입니다.

⸻

📥 입력
	•	첫째 줄에 n, q가 주어집니다.
	•	이후 2^n × 2^n 크기의 격자 정보가 주어집니다.
	•	마지막 줄에 q개의 레벨 정보가 주어집니다.

⸻

📤 출력
	•	첫째 줄에 남아있는 빙하의 총 양을 출력합니다.
	•	둘째 줄에 가장 큰 얼음 군집의 크기를 출력합니다.

⸻

💡 접근 방법
	•	레벨 L마다 2^L × 2^L 크기 블록 단위로 회전
	•	회전이 끝난 뒤 인접한 얼음 개수를 기준으로 동시에 감소 처리
	•	모든 연산이 끝난 뒤 전체 얼음 합 계산
	•	BFS로 가장 큰 얼음 군집 크기 탐색
